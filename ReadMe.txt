exhaust pipes

We created exhaust pipes to the back of the car, with the same color.
We used GLU (helper library) to draw common complex geometric models like Sphere and Cylinder.
First, we set the color of the pipes, second, we initialized Glu, then, for both of them, we draw the Sphere, initializing a local coordinate system, and then we draw the Cylinder.
We used Sphere and Cylinder because they both together form the shape of exhaust pipes.
In the end, we cleared the memory as we learned in REC 8.
